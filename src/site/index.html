Dart suteikia struktūrą web aplikacijų inžinerijai su nauja kalba, bibliotekomis ir įrankiais.

Panašumas

Dart yra objektiškai orentuota, klasėmis paremta kalba. Dart kalba yra lengvai išmokstama daugeliui kūrėjų. Dart padės jums sukurti struktūrizuotą web aplikaciją.

{% highlight dart %} import 'dart:html'; main() { var msg = query('#msg'); var btn = new ButtonElement(); btn.text = 'Click me!'; btn.on.click.add((e) => msg.text = 'Dart!'); document.body.nodes.add(btn); } {% endhighlight %}

Produktyvumas

„Dart Editor“ ir „Dart SDK“ užtikrina integruotą programų kūrimo patirtį. Dart Redaktorius palaiko pertvarkymą ir atskaitos taškus, kodo užbaigimą, kodo atskaitos taškus, kodo navigaciją ir daugiau. SDK sudaro savarankiška virtuali mašina, paketų tvarkyklė, ir Chromium naršyklė su integruotu Dart VM.

Universalumas

Dart gali būti sukompiliuota į JavaScript, todėl jūs galite naudoti tai web aplikacijoms visose moderniuose darbastalio ar mobiliuose naršyklėse. Mūsų JavaScript kompiliatorius garantuoja minimalų kodą tree-shaking technologijos dėka. Dart aplikacijos taip pat gali būti paleidžiamos serveriuose, ant savarankiško Dart VM.
